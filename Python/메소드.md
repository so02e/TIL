## 1. 참조값

* 하나의 리스트 객체를 `공유`하는 경우 같은 참조값(ID)를 갖는다. 어떤 객체를 수정하던지 간에 다른 객체에도 영향을 준다.

  ```python
  l1 = [1,2,3,4,5]
  l2 = l1
  
  print(id(l1)) # 참조값이 동일하다.
  print(id(l2)) # 참조값이 동일하다.
  
  l2[0] = 77
  
  print(l1) # 리스트가 동시에 변했다.
  print(l2) # 리스트가 동시에 변했다.
  ```

  

input은 무조건 문자로 입력받는다.
