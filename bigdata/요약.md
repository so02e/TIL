### 1. 빅데이터 개요 및 활용 ​

#### 1) 데이터와 정보

##### (1) 데이터의 정의와 특징

**데이터**는 <u>추론과 추정의 근거를 이루는 사실</u>로, 다른 객체와의 <u>상호 관계</u> 속에서 더 큰 가치를 갖는다. 과학적 발견은 개인의 암묵적 지식에 근거하는 경우가 많으며, 이를 활용하기 위해서는 암묵지와 형식지의 상호작용이 중요하다.

| 정량적 데이터 |   정성적 데이터   |
| :-----------: | :---------------: |
|     숫자      | 문자, 함축적 의미 |

##### (2) 데이터의 유형

|     정형 데이터     |      반정형 데이터       |       비정형 데이터        |
| :-----------------: | :----------------------: | :------------------------: |
|  정해진 형식. 구조  | 비교적 유연, 연산 불가능 |    구조 X, 연산 불가능     |
| 관계형 데이터베이스 |   JSON, XML, RDF, HTML   | 동영상, 이미지, 문서, 메일 |

##### (3) 데이터의 근원에 따른 분류

|          가역 데이터           |        불가역 데이터         |
| :----------------------------: | :--------------------------: |
| 원본과 1:1의 관계, 추적이 가능 |  가공 데이터, 추적이 불가능  |
| 데이터 마트, 데이터 웨어하우스 | 데이터 전처리, 프로파일 구성 |

##### (4) 지식창조 메커니즘

암묵지와 형식지 간의 상호작용을 위한 일본의 경영학자가 만든 것으로, **①공통화, ②표출화, ③연결화, ④내면화** 의 단계로 구성된다.

데이터 :arrow_right: 정보 :arrow_right: 지식 :arrow_right: 지혜 의 순으로 가치 창출 프로세스가 이루어진다.



#### 2) 데이터 베이스

##### (1) 데이터베이스

데이터 + 기지, 체계적이거나 조직적으로 정의되고 검색할 수 있는 정보의 집합체

① 통합된 데이터

② 저장된 데이터

③ 공용 데이터 : 여러 사용자의 다양한 목적

④ 변화하는 데이터 : 지속적인 갱신

##### (2) 데이터베이스 관리시스템(DBMS)

데이터베이스를 관리하며 응용 프로그램들이 데이터베이스를 공유하며 사용할 수 있는 환경을 제공하는 소프트웨어로 **SQL**언어를 통해 데이터베이스 접근할 수 있다. 

|   관계형 DBMS   | 객체지향 DBMS | 네트워크 DBMS | 계층형 DBMS |
| :-------------: | :-----------: | :-----------: | :---------: |
| 열, 행의 테이블 |     객체      |    그래프     |    트리     |

##### (3) :star: 데이터베이스의 활용:star:

|                      OLTP (transaction)                      |                      OLAP (analytical)                       |
| :----------------------------------------------------------: | :----------------------------------------------------------: |
| 호스트 컴퓨터와 온라인으로 접속된 여러 단말 간 처리 형태의 하나로 데이터 베이스의 **데이터를 수시로 갱신**하는 프로세스 | **정보위주의 분석처리**를 하는 것으로, OLTP에서 처리된 데이터를 분석해 프로세싱하는 것을 의미함 |
|                         데이터 갱신                          |                         데이터 조회                          |
|          동적으로 순간적, 핵심 데이터, 복잡한 구조           |          정적으로 주기적, 주제, 비정규적 읽기, 단순          |

##### (4) 데이터웨어하우스 (DW)

사용자의 의사결정에 도움을 주기 위해 기간 시스템의 데이터 베이스에 축적된 데이터를 **공통의 형식으로 변환**해서 관리하는 데이터베이스로 일정 시간 동안의 데이터를 축적하고 의사결정을 위한 다양한 분석작업을 수행한다.

|   주제 지향성    |          통합성          |    시계열성     |      비휘발성      |
| :--------------: | :----------------------: | :-------------: | :----------------: |
| 중요한 주제 중심 | 일관된 형태, 전사적 관점 | 일정 기간, 시점 | 갱신 이외의 변경 x |

### 2. 빅데이터 기술 및 제도

#### 1) 빅데이터 플랫폼

빅데이터 플랫폼은 수집, 저장, 처리, 분석의 **전과정**을 통합적으로 제공하여 그 기술을 잘 사용할 수 있도록 준비된 환경이다. <br/>비즈니스의 요구사항이 변화하고, 데이터의 규모가 커지고, 구조가 변화하고 유연성이 증대되었기 때문에 등장하게 되었다. <br/>또한, 컴퓨팅의 부하, 저장 부하, 네트워크 부하의 발생을 해결해준다는 점에서 의의가 있고 사용자와 제공자 측면에서 모두 만족시킬 수 있어야 한다.

|                    소프트웨어 계층                     |                       플랫폼 계층                        |             인프라스트럭처 계층             |
| :----------------------------------------------------: | :------------------------------------------------------: | :-----------------------------------------: |
| 어플리케이션을 구성하고 처리와 분석을 위한 수집과 정제 | 실행을 위한 플랫폼을 제공하며, 스케쥴링, 자원할당을 수행 | 빅데이터 처리와 분석에 필요한 자원을 제공함 |

##### (4) 빅데이터 처리

|                         분산 시스템                          |                         병렬 시스템                          |
| :----------------------------------------------------------: | :----------------------------------------------------------: |

`하둡(Hadoop)` 은 분산 처리 환경에서 대용량 데이터 처리 및 분석을 지원하는 오픈 소스 소프트웨어 프레임워크이다. 분산파일 시스템, 데이터베이스, 프레임워크로 구성되어 있다.

`아파치 스파크(Apache Spark)` 는 실시간 분산형 컴퓨팅 플랫폼으로 in -memory 방식으로 처리하여 하둡보다 처리속도가 빠르다. 

`맵 리듀스(Map Reduce)` 는 구글에서 개발한 방대한 양의 데이터를 신속하게 처리하는 프로그래밍 모델로 효과적인 병렬 및 분산 처리를 지원함

#### 3) 빅데이터와 인공지능

`인공지능`은 기계를 지능화하는 노력이며 `지능화`는 객체가 환경에서 적절히 예지력을 갖고 작동하도록 하는 것이다. 인공지능은 합리적 행동이며 어떤 행동이 최적의 결과를 낳을 수 있도록 의사결정 능력을 갖춘 에이전트를 구축하는 것이라 할 수 있다. 인공지능은 설정한 목표를 극대화하는 행동을 제시하는 의사결정 로직이다.

> 인공지능은 기계학습과 딥러닝을 혼재하여 사용한다.<br/>==인공지능== 은 사람이 생각하고 판단하는 사고구조를 구축하려는 전반적인 노력이다.<br/>==기계학습==은 인공지능의 연구분야 중 하나로, 인간의 학습 능력과 같은 기능을 축적된 데이터를 활용하여 실현하고자 하는 기술이다.<br/>==딥러닝==은 기계학습 방법 중 하나로 컴퓨터가 많은 데이터를 이용해 사람처럼 스스로 학습할 수 있도록 인공신경망 등의 기술을 이용한 것이다.



인공지능의 기술동향은 `케라스`는 딥러닝 신경망 구축 인터페이스 라이브러리이고, `GAN`은 두 개의 인공 신경망으로 구성된 딥러닝 이미지 생성 알고리즘이다. `오토인코더`는 라벨이 설정되어 있지 않은 학습 데이터로부터 더욱 효율적인 코드를 표현하도록 학습하는 신경망이다. 



#### 4) 개인정보 

개인정보의 이전은 다른 사람이 개인정보를 처리하는 것, 위탁은 업무를 이전하는 것, 제 3자 제공은 해당 정보를 제공받는 자의 업무 처리를 위해 정보를 이전하는 것이다.



#### 5) 개인정보와 법, 제도

`개인정보보호법`은 당사자의 동의 없는 개인정보 수집, 활용, 제공을 금지하는 법률이다.

`정보통신망법`은 정보통신망의 개발과 보급, 이용 촉진과 함께 통신망을 통해 활용되고있는 정보보호에 대한 법률이다.

`신용정보보호법`은 개인 신용정보를 신용정보회사 등에 제공할 때 공인전자서명이 있는 전자문서에 의한 동의 등을 얻어야 하는 것이다.
